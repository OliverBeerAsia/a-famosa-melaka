## PART 1: TILESET GENERATION

### Master Tileset Brief

Generate a **seamless tileset spritesheet** for a late 90s isometric RPG. The tileset should be arranged in a grid format, with each tile being **32x32 pixels** (or 64x64 for high-detail versions that will be downscaled).

**Art Style Requirements**:
- Pixel art with visible but refined pixels
- Rich color palette (48-64 colors maximum)
- Subtle dithering for texture and shading
- Warm tropical lighting (golden hour feel)
- Slight weathering and wear on all surfaces
- Consistent light source from top-left

## GENERATION NOTES

### For Best Results:

1. **Specify pixel dimensions** - Be exact: "32x32 pixels" not just "small"
2. **Reference specific games** - "Ultima VII style", "Ultima VIII density"
3. **Color palette** - Provide hex codes for consistency
4. **Lighting direction** - Always specify (usually top-left)
5. **Request seamless tiling** - For ground tiles especially
6. **Ask for sprite sheets** - More efficient than individual sprites
7. **Emphasize the era** - "Late 1990s" triggers the right aesthetic

### Technical Specifications:

- **Tile Size**: 32x32 pixels (will be scaled to 16x16 in-game for authentic retro feel)
- **Character Size**: 24x32 pixels per frame
- **Scene Size**: 960x540 or larger (will scale to fit)
- **Color Depth**: 8-bit indexed color aesthetic
- **Format**: PNG with transparency where needed
